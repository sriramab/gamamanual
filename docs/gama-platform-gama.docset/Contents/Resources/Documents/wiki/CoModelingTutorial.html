Co-modeling tutorial

This tutorial introduces the co-modeling feature offered by GAMA.

Model Overview

Step List

This tutorial is composed of 3 steps corresponding to 3 models. For each step we present its purpose, an explicit formulation and the corresponding GAML code.

  1. Presentation of the weather and plant growth models
  2. Instanciate a weather model and step it
  3. Display agents and indicators of the weather model
  4. Run 2 co-models in a single model
  5. Run 4 plantgrow model coupled with a weather model
